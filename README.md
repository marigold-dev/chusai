# Chusai

# Developer instructions
## Prerequisites
### Ligo compiler 
Minimum Ligo compiler version is 0.41.0

Installation instructions can be found [here](https://ligolang.org/docs/intro/installation)

Please check your Ligo instalation with 
```
$ ligo --version
0.41.0 
```

## Building
```
$ make build
```

## Testing
```
$ make test
```
